# Project Background

This project is based on my data and operations work for a real charity sale
that raised over ¥26,000. My work was mainly on the preparation and review
side, rather than being the only person responsible for the whole event.

During preparation, I helped count donated items, organize them into
categories, estimate item values, maintain online spreadsheets, support item
allocation, help plan booth layout, and track preparation progress. After the
event, I organized an anonymized version of the records and used Python to
summarize donations, item categories, sale revenue, team contributions, and
booth-level results.

The data in this repository is anonymized and sample-based. Real names, phone
numbers, school names, organization names, and private donor information are
not included.
