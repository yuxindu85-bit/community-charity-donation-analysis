# Project Background

This project is based on my data and operations work for a charity sale that
raised over 26,000 CNY.

During preparation, I helped count donated items, organize them into categories,
estimate item values, maintain online spreadsheets, support item allocation,
help plan booth layout, and track preparation progress.

After the event, I organized an anonymized version of the records and used
Python to summarize donation records, sale records, item categories, and team
contributions.

All private names, donor information, phone numbers, school names, and
organization names were removed or anonymized before the data was used in this
repository.
