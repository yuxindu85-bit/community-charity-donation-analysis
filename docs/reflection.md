# Reflection

Before this project, I thought charity sale work was mostly about collecting
items and selling them. After working on the data side, I learned that
organization and record keeping are also important.

Clear data helped with item allocation, booth planning, progress tracking, and
post-event review. If item records are unclear, it becomes harder to decide
where items should go, which team is responsible for them, and what needs to be
checked after the event.

This project helped me connect community service with data analysis. The code is
not complicated, but the process taught me how useful simple tables, charts, and
tests can be when a real activity has many moving parts.
