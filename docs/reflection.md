# Reflection

Before this project, I thought charity sale work was mostly about collecting
and selling items. After working on the data side, I learned that organization,
clear records, and simple analysis can make a real event easier to manage and
improve.

The most useful part for me was seeing how different records connected to each
other. Donation records helped show where direct support came from. Inventory
records helped with item allocation and booth planning. Sale records helped
review which categories performed better. Booth records helped explain why some
areas needed more tables or clearer item grouping.

This project helped me connect community service, operations, and data
analysis. It also reminded me that a useful project does not need to be overly
complicated. Clear data, careful checking, and honest reflection can already
make the work more meaningful.
