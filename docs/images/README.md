# Activity Evidence Image Folder

This folder is reserved for privacy-protected activity evidence images.

Only blurred images should be added here. Before any event photo is uploaded,
visible faces, private names, contact details, QR codes, payment information,
school names, organization names, and sensitive text should be removed or
blurred.

Suggested filenames:

- `charity_sale_booth_blurred.png`
- `toy_booth_setup_blurred.png`
- `event_stage_performance_blurred.png`
- `dashboard_overview.png`

The actual blurred activity images are not included in this repository until
they have been checked for privacy.
