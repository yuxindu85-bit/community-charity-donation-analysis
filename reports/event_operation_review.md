# Event Operation Review

## Purpose

This review focuses on how data supported the operations side of the charity
sale.

## Item Allocation

Inventory records helped group donated items by category, quantity, estimated
value, condition, team, and booth area. This made it easier to decide where
items should be placed before the event.

## Booth Layout

Booth layout records helped compare planned item counts with actual item
counts. Larger categories and smaller fast-moving items needed more visible
space, especially on the shared table.

## Preparation Tracking

Online spreadsheets helped track which teams were responsible for different
item groups. This reduced confusion during preparation and made it easier to
review missing or incomplete records.

## Team Coordination

Team-level summaries combined donation and sale records. This helped review
each team's contribution without relying only on memory after the event.

## Post-Event Review

After the sale, the cleaned data made it possible to review category revenue,
booth revenue, item quantities, and estimate vs actual price differences. These
results can help improve future charity sale planning.
