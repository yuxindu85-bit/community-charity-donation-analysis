# Community Charity Donation Analysis

## Project Overview

This project analyzes anonymized donation, inventory, and sale records from a
charity sale that raised over 26,000 CNY. I supported the data and operations
side of the event by organizing item records, estimating values, maintaining
spreadsheets, helping with item allocation and booth planning, tracking
preparation progress, and reviewing post-event data.

The goal of this repository is to turn those records into a small Python data
analysis project. It is not an official financial audit. It is a cleaned and
anonymized learning version of the event data.

## My Role

My work was mainly on the data and operations side:

- Counted donated items and grouped them into categories.
- Estimated item values to help with price planning.
- Maintained online spreadsheets before and after the event.
- Helped match items with teams and booth areas.
- Tracked preparation progress before the sale.
- Organized post-event records for review.

## Key Results

Using the sample data in this repository:

- Direct donations: 23,090 CNY
- Charity sale revenue: 4,821 CNY
- Combined total: 27,911 CNY
- Highest sale category by revenue: Handmade Crafts
- Highest combined team contribution: Team A

Direct donations were larger than sale revenue. This makes sense for this event
because several donors gave larger one-time amounts, while most sale items were
lower-priced donated goods.

## How to Run

Install the packages:

```bash
pip install -r requirements.txt
```

Clean the raw CSV files:

```bash
python src/clean_data.py
```

Create summary tables and the report:

```bash
python src/analyze_charity_sale.py
```

Create charts:

```bash
python src/create_charts.py
```

Run the tests:

```bash
python -m unittest discover -s tests
```

## Data Privacy Note

All names, phone numbers, school names, and organization names were removed or
replaced before the data was added to this repository. Donors are shown with
anonymous labels such as `Donor_001` and `Business_001`. The dataset is for
learning and project documentation only.

## What I Learned

Before this project, I thought charity sale work was mostly about collecting
items and selling them. After working with the data, I learned that record
keeping is also important. Clear item records helped with booth planning, team
allocation, price discussion, and post-event review.

I also learned that a simple data project still needs careful checking. If the
CSV files are messy or the totals are not tested, the charts can look correct
even when the numbers are wrong.
